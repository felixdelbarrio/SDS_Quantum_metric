# Datasets Runbook

## Audit

1. Open Datasets.
2. Pick the country card.
3. Use the entity tabs to inspect RAW calls, visual contracts, snapshots, derived datasets, chart payloads, regression results, discrepancies and manifest data.

Datasets is an audit console. It does not expose manual derived build, regression or audit actions.

## Export

Use the export icon for a country. The ZIP includes Parquet and manifest data. It does not include cookies, tokens or secrets.

## Import

Use Importar with a ZIP generated by the application. The backend validates `manifest.json` and only imports Parquet under the active persistent data directory.

## Delete

Deletion requires:

```http
DELETE /api/datasets/{country}?confirm={country}
```

The modal requires the exact country code. Deletion removes RAW, derived datasets, visual contracts, web snapshots, regression datasets and ingestion history for the country.
