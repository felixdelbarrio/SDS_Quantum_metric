# Product Spec

La app permite configurar Quantum, probar conexion real, ingerir datos reales, persistirlos por pais y consultar offline KPIs, tablas y llamadas capturadas.

Secciones:

- Home: resumen de datos locales.
- Quantum: configuracion no sensible y test.
- Ingesta: nueva ingesta, progreso, historial, cancelar.
- Datasets: paises, volumen, import/export/limpieza.
- As-Is Explorer: inventario documentado de rutas, APIs y widgets.
